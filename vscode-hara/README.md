# Hara VS Code Extension

This extension connects to the Hara server using the Redis protocol.

## Features

- **Connect to Server**: Connect to a running Hara server.
- **Evaluate Selection**: Evaluate selected code on the server.

## Configuration

- `hara.server.host`: Hostname of the Hara server (default: 127.0.0.1).
- `hara.server.port`: Port of the Hara server (default: 4164).
- `hara.session`: Session to use for evaluation (default: ROOT).

## Requirements

- A running Hara server instance.
