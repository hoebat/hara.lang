# The Little Book of HAL

Status: **placeholder** — planned, not yet written.

## Intent

A gentle introduction to HAL (the hara language) in the style of
*The Little Lisper*: a series of short question-and-answer dialogues
that teach the language by doing, one small idea at a time.

Planned shape:

- Q&A/dialogue format — each chapter asks, the reader answers, the book
  confirms.
- Examples run as-is in the hara REPL (`./hara repl` from the repo
  root, or the browser playground on the website).
- Progression: data → functions → collections → namespaces →
  state and effects → extensions.
